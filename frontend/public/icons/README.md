This folder contains PWA icons.

Currently included:

-   clinic-icon.svg (source)
-   clinic-icon-192.png (raster for iOS/Android)
-   clinic-icon-512.png (install prompt)
-   clinic-icon-maskable-512.png (maskable)

If PNGs are missing, export from the SVG using any editor (e.g.,
Inkscape/Figma):

-   192x192 PNG from clinic-icon.svg
-   512x512 PNG from clinic-icon.svg
-   For maskable variant, ensure safe padding (48px) inside 512x512 and export
    as clinic-icon-maskable-512.png
